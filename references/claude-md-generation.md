# CLAUDE.md 생성 프로토콜

**폴더 스캔 통합:** CLAUDE.md + GRAPH.md 동시 요청 시 폴더 스캔 1회만 실행. 스캔 결과를 공유하여 CLAUDE.md 생성 + GRAPH.md 생성에 재사용. 순차 요청이더라도 같은 세션 내면 이전 스캔 결과 캐시 활용.

```
①인터뷰(유형판별+정보수집) → ②템플릿 로드 → ③폴더 스캔(선택) → ④초안 생성 → ⑤형 검토·수정 → ⑥저장
```

## ① 인터뷰

**발동 즉시 인터뷰 프로토콜을 실행.** `→ interview.md 참조`

- **1단계**: AskUserQuestion 1회로 유형 판별 3개 질문 동시 수집
- **2단계**: 유형별 필수 질문 (A=4개, B=2개, C=2개) 수집 → 필수 답변 모이면 즉시 초안 생성
- **"그냥 빨리 만들어"**: 폴더 스캔 + 최소 추론으로 C유형 초안 즉시 생성

## ② 템플릿 로드

`→ templates.md 참조`

## ③ 폴더 스캔 (선택)

마운트된 프로젝트 폴더가 있으면: 파일 목록·frontmatter·_sources/ 수집 → SoT 섹션 자동 채움

## ④ 초안 생성

| 규칙 | 이유 |
|------|------|
| 인터뷰 답변 있는 항목 반드시 채움 | [미확정] 남기면 인터뷰 의미 없음 |
| 추론 불가 항목만 `[미확정]` | 빈 플레이스홀더 금지 |
| 가정 가능하면 가정 명시 후 채움 | 빈 칸보다 가정+채움이 낫다 |
| 200줄 이내 | A4 2장 한도 |
| `[미확정]` 5개 초과 시 경고 | 양식이 아니라 지침이어야 한다 |
| 파일명: `CLAUDE.md` 고정 | 프로젝트 폴더 루트 |

## ⑤ 형 검토 → ⑥ 저장

형 승인 후 저장. 저장 후 1줄 경로 보고 필수.
